# Codex Skills

This repository contains reusable Codex skills. Each skill is a self-contained folder with a `SKILL.md` plus optional scripts and references.

## Included Skills

- `ue-build-assistant`: Unreal Engine code + build assistant.
- `github-project-builder`: Read a project, create the environment, build, and run tests.

## How to Install

Use the `skill-installer` skill to install from this repo URL:
`https://github.com/UnrealXu/codex-skills`

## Add New Skills

Add a new folder under this repo, then commit and push. Each skill must include a `SKILL.md` with YAML frontmatter.
